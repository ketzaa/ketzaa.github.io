แฟ้มสะสมผลงาน (Portfolio)

ยินดีต้อนรับสู่พื้นที่แบ่งปันโค้ดและผลงานของฉัน

เกี่ยวกับฉัน

สวัสดีครับ/ค่ะ ฉันเป็นนักเรียนที่หลงใหลในการเขียนโปรแกรมและการพัฒนาซอฟต์แวร์ เว็บไซต์นี้สร้างขึ้นเพื่อรวบรวมโปรเจกต์และโค้ดต่างๆ ที่ฉันได้เรียนรู้และสร้างสรรค์ขึ้นมา

ผลงานและโค้ด (Code Sharing)

📝 โปรเจกต์: โปรแกรมคำนวณเกรดเฉลี่ย (Python)

โปรแกรมสำหรับคำนวณเกรดเฉลี่ยจากคะแนนสอบ เขียนด้วยภาษา Python เน้นการใช้ Logic พื้นฐานและฟังก์ชัน

ดูโค้ดบน GitHub

🌐 โปรเจกต์: เว็บไซต์ส่วนตัว (HTML/CSS)

เว็บไซต์ที่ออกแบบด้วย HTML และ CSS เพื่อใช้เป็นแฟ้มสะสมผลงาน โค้ดอ่านง่ายและเป็น Responsive
